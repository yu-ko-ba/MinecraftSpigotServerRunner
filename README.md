# MinecraftSpigotServerRunner

Spigotを使ったマインクラフトサーバーを起動するためのシェルスクリプトです
このシェルスクリプトが置かれているフォルダに移動してからjarファイルを起動するので、実行するフォルダを間違えて新しくワールドが作られるのを防げます(｀・ω・´)
